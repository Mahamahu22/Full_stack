3. What is Docker Compose?
Docker Compose allows you to run multiple containers at once using a single file (docker-compose.yml).

 Example: Node.js App + MongoDB

docker-compose.yml
-------------------
version: '3.8'
services:
  app:
    build: .
    ports:
      - "3000:3000"
    depends_on:
      - mongo
  mongo:
    image: mongo
    ports:
      - "27017:27017"

Run everything:
docker-compose up
